Tutorial - How to simulate a portofolio

4. Add a discount to a tariff

Every tariff fee of a tarification simulation can be slightly modified to allow localized or negociated discounts on this specific tarification. For example, a discount could be applied to transaction fees on a particular fund.

Step by step process

4.a. Adding a discount



First, you must do a right click (or a double click) on the tab of the worksheet you wish to apply the discount on.



Then launch the discount editor by clicking on the link Edit discount.

If this worksheet didn't have a discount already, the discount editor will have a button that allows you to create a discount at this point.



Once the button is clicked, the discount editor will change to a screen that allows you to enter a discount rate, and if needed, a minimum fee with or without a maximum fee.





Once the window closed, changes will be taken in account in the simulation.

4.b. Tracking discounts in the Tariff distribution tree



As you can see on next picture, you can track down any discounts that has been added to a specific portfolio by looking to the Tariff distribution tree.
Discounts are found on tariffs which are tagged by a little orange cross in the upper right corner. This property is reflected up to its higher abstraction, therefore allowing to find tariffs even when the tree is collapsed.



4.c. Modifying the way of viewing discounted fees



You can modify the way you wish to see the portfolio fees. The preferences are available through a menu found at the top of the Simulator window : "Discount". As shown on picture below.



The three options are :
  • Hide discounts informations. This is the default view : you see only one fee column which include discounts (1)
  • Show discounts. This will spawn a second column which content will correspond to the amount of the discount for each position (2)
  • Show un-discounted fees. This will spawn a second column which content will correspond to the fees that would have been charged if no discount applied (3)
