Tutorial - How to simulate a Portfolio

1. Open a new Simulation

TariffEye allows you to simulate a tarification, introducing values for the assets of a portfolio, transaction amounts or futures in the different workplaces defined by the tarification document.

Step by step: Create a new simulation

  1. In the Desktop window, choose New simulation in the Simulation menu; or click on New portfolio... quick link in the Startup window .



  2. A file chooser dialog appears. Select the Tarification file (extension .tye) you want to use to simulate. By default, the tarificaiton library directory is listed. But you can also select a tarification file from your own work, using the button Use a user's tarification file. Validate your choice using the open button.

  3. The Simulator window apprears. You can now simulate your Tarification.
