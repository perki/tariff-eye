Tutorial - How to simulate a portofolio

5. View simulation results on the graphical user interface

TariffEye allows to view realtime changes made to the portofolio.

Read results

On the following screenshot you can distinguish different parts of the user interfaces that allow to read the results.


The column at the right contains the costs for each position of the portfolio. In the assets branch you will see mostly custody fees, whereas in the transaction section you will find costs induced by transactions over those positions.
In (a) which is represented the total costs of this portfolio.
(b) which represents the custody fees associated to all equities.
You can see that all these amounts are in swiss francs in (c).
If you wish, you can at any moment hide (or show) the costs column by pressing the button (d).

If you use the Tree orderer tool (see manual) which is accessible in the Options menu, you can then organize the view with a different splits. For example you could prioritize the localization to view the cost repartition by countries.
