Tutorial - How to simulate a Portfolio

2. Fill-in your investment portfolio

To simulate your tarification, you will need to assign amount of money in the different allocation Positions. There's two ways to do this, you can either browse all Tariffs and enter the appropriate amount of money, the list of transactions, or the set of futures for each of them; or you can use the automatic distribution feature of TariffEye. The following sections describe these methods.

2. a: Manually assigning values

Step by step:

  1. Choose a Tariff in the Tarif distribution tree. The content appears in the window.


  2. Add an asset value, a transaction or a future, using the add button.


  3. Enter the desired values in the right fields.


  4. Repeat steps 1-3 for all Tariffs.
  5. The resulting fees, and the resulting sum of assets is shown in the simulator window.

2. b: Automatic distribution of an amount in the Assets Workplaces

Step by step:

  1. Select an asset position in the Tariff distribution tree. The window shows a pie chart, representing the money amounts distribution between all nested positions. E.g. If you select the Assets root position, the pie chart contains the distribution of your assets between bonds, funds, warrents and so on. The total amount of assets for the selected position apears in the amount field.
  2. Modify the value of assets total. TariffEye distributes the new amount, according to the old distribution.

2. c: Automatic turnover computation in the Tarification Workplaces

You can also specify the turnover or rotation of your transactions. TariffEye computes the resulting number of lines and the amount for the transactions. To do so follow these steps:

Step by step:

  1. Select a transaction position in the Tariff distribution tree. The window shows the distribution between the nested positions of the selected item.
  2. The turnover field indicates the current turnover for the selected positions. The link icon indicates if the turnover is linked with the enclosing position, or if it can be modified for this position. To change the state of the link, click on the link icon.
  3. When the link is broken, you can enter a new value for the turnover. TariffEye computes the new values for your transactions.



  • The turnover is calculated as follow : For 200'000 CHF of a specific asset, a turnover of 50% would generate sales for 100'000 CHF and purchases for 100'000 CHF.


2. d: Modify the distribution rates between the nested positions

You can modify the distribution rates shown on the pie chart for a position. To do so, follow these steps:

  1. Select a position in the Tariff distribution tree. The distribution of the nested position is shown in the pie chart. A slide bar for each nested position is shown, representing the current relative distribution rate.


  2. Modify the distribution by moving the slide bar. The value of the other positions are recomputed according to their old relative distribution. To avoid changing the value of another position when you move a slide bar, lock the position using the lock button . Only the values of the unlocked positions are then modified.

2. e: Advanced distribution options

There's several options specifying the way TariffEye distributes the amounts and the number of lines. Access the distribution option panel using the expand button .
For the assets distribution and transaction turnover, you can specify a preferred number of lines. For the assets distribution, you can specify either the number of lines or the size of a line. To do so, follow these steps:

Step by step:

  1. Select a position in the Tariff distribution tree.
  2. Open the distribution option panel using the expand button.
  3. For the assets distribution, choose one of the following in the menu:
    • Relies on parent: the distribution of the position is defined by the enclosing one.
    • Set the number of lines: you can specify manually the number of lines for all nested asset positions.
    • Set the preferred size of a line: you can specify manually the amount of the lines of all nested asset positions.
  4. For the transaction distribution, choose one of the following:
    • Relies on parent: the distribution of the position is defined by the enclosing one or higher-level "parent".
    • Set the number of lines: you can specify manually the number of lines of all nested transaction positions.

2. f: Fee calculation

Once the amounts are distributed into the workplaces, TariffEye computes all fees applied by the particular bank tarification on your different positions. You can view these values in the cells of the Tariff distribution tree.
