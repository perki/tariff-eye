3. The Tree orderer window

TariffEye allow you to reorder types to view costs repartition from a different angle.

In simulation mode choose the Tree Orderer from the Options menu.



This will open a small window allowing you to reorder financial tools by a different classification. For example you could prioritize localization over types of funds.


To do so, you have to drag one of the entity to a new position in the list. Dragging Base tools under localisation would prioritize localization over standard financial tools definition.
After this operation, the tree representing the portfolio will reconstruct itself to incorporate the changes
