Presentation of the Graphical User Interface

2. The simulator window

One simulator window is opened for each opened simulation. The following parts compose it:



pic 1

  1. Tariff distribution tree: It is the navigator part, where you can browse the positions of your tarification.
  2. Fee cells: They contain the simulated fee for each of the positions of the Tariff distribution tree.
  3. Simulator panel: It shows the content of the selected position.

2. a: Tariff distribution tree

The Tariff distribution tree (part (a) on pic 1) shows the list of positions you can simulate. It uses a tree representation, i.e. some of them are nested. You can open and close the position using the expand button to show or hide which positions are nested in the selected one:

The differents colours of the positions represent the different kinds of information the position gives, e.g. localization information, financial information and so on. The order used to sort the node is defined by the tree orderer. For further information about the position properties and the color code, see the chapter about the tree orderer (3).

As you can see, there are two different icons to represent a position. The icon means that the position contains many tariffs (each of them defined by other position information), then it can be opened to see the nested positions. The icon means the position contains a single tariff.

2. b: Fee cells

For each position of the Tariff distribution tree one value is associated in the corresponding fee cell (part (b) on pic 1). This cell contains the value of the perceived fee for the position (and all nested positions).

2. c: Simulator panel

The simulator panel (part (c) on pic 1) shows the contents of the selected position in the Tariff distribution tree. You can simulate your portfolio by changing the values in this panel.
For further information about the simulator and how to use it, please refers to the tutorial "How to use the simulator".

2. d: File menu

Close End the current simulation.
Save Save your changes in the Portfolio file.
Save as... Save your changes in a new Portfolio file.

2. e: Reports menu

The items of this menu yield to the production of reports.

Fee report summary... The fee report contains all the values simulated by the portfolio. The summarized version don't show the content of each position, but only the fee perceived.
Fee detailed report... The fee report contains all the values simulated by the portfolio.
Tarification report... The tarification reports contains information about the tarification used by the simulator.

2. f: Options menu

Tree orderer... Show the Tree orderer window.

2. g: Import menu

The import feature of TariffEye allows to compare two tarifications. When you are simulating a portfolio, you can import another portfolio file in it, the values of the imported portfolio will then either be summed to the current ones or replace them. Note that you can import a portfolio in another one even if they use different tarification.

From file... Import a portfolio using a portfolio file.
From documents at work Import an already opened portfolio.

2. h: Data menu

Remove all user options Reset the simulation.
