Presentation of the Graphical User Interface

1. The Desktop window

The Desktop window is the main window of the software. All your work will appear on it, using internal windows.


The menu bar of the Desktop window contains the following items:

1. a: File menu

Manage currencies... Open the currency manager window,
Update tariffs database Open the tariff update tool
Assistant Open the quick launch window
Themes Allow to choose the look and feel of the software.
Quit Exit the software.

1. b: Creation menu

This menu is only aviable on the full version

This menu accesses the Tarification creator. With this tool, you can create and edit tarification files.

New tarification Create a new empty tarification
Open tarification... Open an existing tarification from a tarification file.

1. c: Simulation menu

This menu accesses the Tarification simulator. With this tool you simulate portfolios over your tarification files.

New simulation Create a new simulation, using a tarification file
Open portfolio... Open an existing simulation, from a portfolio file.

1. d: Tools menu

Preferences Open the preferences window.

1. e: Window menu

In this menu is the list of all windows shown in the Desktop. You can focus on them choosing the coresponding item in the menu. In top of the menu you find also the following items:

Cascade windows Organize the window in a cascade form. All windows are superimposed.
Tile windows Organize the window in a tile form. All windows are resized and visible.

1. f: Help menu

TariffEye help... Open the help window
About... Open the About dialog.
