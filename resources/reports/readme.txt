This folder contains all templates that are used
while creating JasperReports reports. 

reports
   +------templates   All JasperReports XML templates
   +------compiled    The compiled templates (automatically
                      updated from templates directory)

The only files that can be modified by hand are 
those under ./templates. You should use iReports or
a similar tool for easy editing.
