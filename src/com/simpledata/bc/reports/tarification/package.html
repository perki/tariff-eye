This package contains the classes used to build a tarification report.

This report is generated by the TarificationReportFactory, who in turn uses CompactNodeLinearizer to build sections and subsections and TarifReportConstructor to build single section content.

All the classes that start with Subreport* are representing classes that can be found in the datamodel or in the components to the report world. View classes.
